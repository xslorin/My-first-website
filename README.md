# Своя игра: 23 февраля

## 📖 Описание

**Своя игра: 23 февраля** — мой первый сайт, который я разработал к празднику **День защитника Отечества**. Проект представляет собой небольшую викторину в стиле телевизионной игры «Своя игра», где пользователь отвечает на вопросы, посвящённые истории, военной технике, известным личностям, героям Второй мировой войны и культуре.

Во время разработки я познакомился с основами **HTML**, **CSS** и **JavaScript**, научился работать с формами, хранением данных в браузере и созданием интерактивных страниц.

---

## ✨ Возможности

- Регистрация пользователя перед началом игры.
- Сохранение имени, фамилии и группы через `localStorage`.
- 5 тематических викторин.
- По 15 вопросов в каждой категории.
- Таймер на прохождение каждой викторины.
- Начисление очков за правильные ответы.
- Вывод объяснения после неправильного ответа.
- Автоматический подсчёт общего количества очков.
- Отображение итоговой оценки.
- Просмотр информации о зарегистрированном пользователе.

## ⚙ Как работает игра

1. Пользователь проходит регистрацию.
2. Информация сохраняется в браузере.
3. На главной странице отображаются данные пользователя.
4. Пользователь выбирает одну из категорий.
5. Запускается викторина.
6. На каждый вопрос можно выбрать один из трёх вариантов ответа.
7. При правильном ответе начисляются очки.
8. При ошибке выводится правильный ответ и объяснение.
9. После прохождения категории результат сохраняется.
10. После завершения всех викторин отображается общий результат.
